    • LeetCode14. Longest Common Prefix

    Write a function to find the longest common prefix string amongst an array of strings.

    暴力穷举

    先找出最短的一个字符串，然后遍历这个字符串的每个字符，与其余每个字符串对应的位置做比较，直到找到不同时中止，时间复杂度O(N*M)
